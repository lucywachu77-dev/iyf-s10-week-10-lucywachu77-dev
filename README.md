## Project Description
A RESTful API built using Node.js and Express for managing posts in a CommunityHub system. It supports full CRUD operations.

## Technologies Used
- Node.js
- Express.js
- CORS

## Features
- Get all posts
- Get single post by ID
- Create new post
- Update existing post
- Delete post
- Modular structure (routes + controllers)

## How to Run
1. Install dependencies:
   npm install

2. Start server:
   node server.js

3. Open:
   http://localhost:3000

## API Endpoints

GET /api/posts  
GET /api/posts/:id  
POST /api/posts  
PUT /api/posts/:id  
DELETE /api/posts/:id  

## Lessons Learned
Learned how to build a REST API using Express, structure code using controllers and routes, and handle HTTP methods.

## Challenges Faced
Understanding how to separate logic into controllers and routes, and debugging server connection issues.
